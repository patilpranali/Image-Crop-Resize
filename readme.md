
# Image Crop Resize and Add the Enrty to DB
This project is all about letting User choose the Image to crop, display it in required size (reduced size of 1000px width), let User crop the Image and once submitted send it to the server via AJAX where the picture will be saved in a folder and entry of Image will be done in sql server.

## YOu need:
- Web Server
- mySql DB

## Instructions:

1. Deploy the application to a web server with PHP enabled
2. Update the database connection details in myapp.ini
3. Open the index.html from the web server URL
4. Follow the video (video.webm) to select, crop & upload image. You can crop multiple images one after the other.
